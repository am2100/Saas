Tournament.rb

A Rock/Paper/Scissors tournament.

Takes an array of ["Player", "Strategy"] arrays. The Array can be nested to an arbitrary depth. Array must contain all valid strategies, and contain n^2 players or a validation error will be thrown.

a1 = [[[["Robin", "R"], ["Jim", "P"]], [["Jacqui", "S"], ["Pierre", "r"]]], [[["Zebedee", "p"], ["Sylvan", "s"]], [["Elly", "R"], ["Wilf", "P"]]]] t = Tournament.new(a1)
